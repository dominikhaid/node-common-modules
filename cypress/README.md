## Description

Basic [Cypress](https://docs.cypress.io/) setup to play around with.

## Tasks

### install

Installs Cypress and all needed Dependencies.

```bash
npm install
```

### cy:open

Run Cypress App.

```bash
npm run test:cy:open
```

### cy:run

Run Cypress in CLI.

```bash
npm run test:cy:run
```
